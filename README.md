# React To-Do List App

A simple and interactive To-Do List application built with React.

## Features

* Add new tasks
* Delete tasks
* Move tasks up and down in the list
* Prevent empty tasks from being added
* Responsive and easy-to-use interface

## Technologies Used

* React
* JavaScript (ES6+)
* HTML
* CSS

## How It Works

Users can enter a task in the input field and click the **Add** button to add it to the list.

Each task includes:

* **Delete** button to remove the task
* **Up** button to move the task higher in the list
* **Down** button to move the task lower in the list

The application uses React's `useState` hook to manage the task list and input field state.

## Installation

1. Clone the repository

```bash
git clone https://github.com/Mayoorthanan/To-Do-List.git
```

2. Navigate to the project folder

```bash
cd todo-list-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

or

```bash
npm start
```
